# walmart-sales-demand-forecasting
Machine Learning based Walmart Sales Demand Forecasting using Python and Flask
This project predicts future weekly sales demand for Walmart departments using historical sales data and machine learning.

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features
- Sales demand prediction
- Graph visualization
- Web-based interface

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the app:
   python app.py
3. Open browser:
   http://localhost:5000
